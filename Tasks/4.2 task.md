В с не будет присвоени ничег оиз-за несовпадения типов. 
```python
>>> a = 1
>>> b = "2"
>>> c = a + b
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: unsupported operand type(s) for +: 'int' and 'str'
```

Получить 12:
```python
>>> c = str(a) + b
>>> print(c)
12
```

Получить 3:
```python
>>> c = a + int(b)
>>> print(c)
3
```
